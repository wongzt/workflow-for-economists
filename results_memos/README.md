# Results Memos

This directory stores interpretation memos that are distinct from submission QA reports.

Use subdirectories:

- `theory_audits/` for theory-facing interpretation audits
- `story_diagnostics/` for memo-driven next-step loops when the economics story is not yet settled

These memos are not a substitute for `quality_reports/`. They are for result interpretation, disciplined empirical iteration, and explicit stop decisions.

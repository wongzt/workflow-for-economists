# Editor-Desk Reject Gate

Use this card near the end of the workflow to pressure-test the strongest questions before presenting them as mature candidates.

Sources synthesized from the existing cards: Heilmeier Catechism, strong inference, problem choice, problematization, and problem picking.

## Core Idea

Before a question becomes a recommendation, make the strongest rejection case against it. A mature research question should survive a skeptical editor, reviewer, PI, or committee member who asks why this question matters now and why the proposed evidence can answer it.

## Desk-Reject Checks

Reject, rewrite, or park a candidate if it has any fatal flaw:

1. **No stake:** It does not change theory, method, practice, policy, or a next decision.
2. **Novelty-only:** The main argument is that no one has done it.
3. **Method-first:** The method is clearer than the problem.
4. **No rivals:** It protects one explanation instead of comparing plausible alternatives.
5. **No falsifier:** No realistic evidence could weaken or revise it.
6. **No feasible entry:** A credible pilot cannot start with available resources.
7. **No downside learning:** A negative result would only be disappointing, not informative.
8. **Wrong audience:** The people who should care would not recognize the question as meaningful.
9. **Unclear measurement:** The key construct, outcome, or comparison cannot be observed well enough.

## Output Format

```markdown
## Editor-Desk Gate

**Candidate:** ...
**Likely rejection:** ...
**Fatal?** yes / no / uncertain
**Repair:** ...
**Decision:** advance / rewrite / park / discard
```

## Repair Moves

- Add a rival hypothesis.
- Narrow to a sharper boundary condition.
- Replace novelty with consequence.
- Replace a method with the uncertainty it resolves.
- Add a kill criterion and a two-week pilot.
- Change the audience or target venue.

## Red Flags

- The agent presents every candidate as viable.
- Objections are generic rather than tied to the candidate.
- Repair moves make the project bigger instead of sharper.
- The final recommendation hides major risks.

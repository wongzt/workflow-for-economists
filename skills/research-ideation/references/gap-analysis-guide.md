# Gap Analysis Guide

Do not call something a gap just because few papers use the same setting.

## Strong Gap Types

- the literature disagrees on the sign, magnitude, or mechanism
- a policy-relevant margin is understudied because existing data were weak
- identification is weak in the classic papers and a stronger design is now feasible
- a known effect exists, but the welfare-relevant mechanism is unresolved

## Weak Gap Claims

- "no one has studied this exact city"
- "few papers combine these two keywords"
- "this topic is popular so another paper would be useful"

## Gap Checklist

1. What does the literature already know?
2. What does it still not know?
3. Why has that question remained unresolved?
4. What allows this project to answer it more credibly?
5. Is the contribution new data, cleaner identification, a sharper mechanism, or a new institutional setting?

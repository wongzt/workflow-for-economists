# Zotero Integration Guide

Use Zotero as the citation backbone for economics projects.

## Recommended Collection Structure

```text
Project-[topic]
  |- Core Papers
  |- Identification
  |- Data And Context
  |- Mechanisms
  |- To Read
```

## Import Priority

1. DOI or publisher URL
2. working-paper URL from `NBER`, `IZA`, `CEPR`, or `SSRN`
3. manual metadata cleanup only after a reliable source page is found

## Good Practice

- keep working-paper and published versions distinct
- tag papers by identification strategy
- tag papers by dataset or institutional setting
- export verified citations to `BibTeX` only after metadata cleanup

## What To Avoid

- importing from memory
- treating search snippets as final metadata
- mixing working-paper and journal versions under one vague record

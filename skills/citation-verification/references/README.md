# Citation Verification Reference Files

These files document the citation-verification workflow and common economics-specific failure modes.

## Files

- `common-errors.md`: typical citation problems in economics papers and revision letters
- `verification-rules.md`: versioning and metadata rules
- `api-usage.md`: example API usage for DOI-first verification

## Main Rule

Use DOI and publisher metadata first. Use working-paper series metadata second. Use broader search tools only to discover or cross-check.

# API Usage

## Preferred API Order

1. Crossref for DOI metadata
2. OpenAlex for discovery and cross-checking
3. optional broader scholarly APIs when the first two are insufficient

## Use Cases

- verify DOI metadata
- match title, authors, and year
- distinguish published article from working-paper version
- fetch stable URLs for bibliography cleanup

## Rule

APIs support verification. They do not replace reading the actual source page when the paper relies on a specific claim.

# Regression Table Conventions

- one dependent-variable family or one specification family per column set
- use bottom rows for `Controls`, `FE`, `Clustered SE`, and weights when relevant
- state the clustering level in the note
- disclose sample restrictions clearly
- if controls are added, avoid silently changing the sample
- for `IV`, separate reduced form, first stage, and second stage into readable panels or separate tables
- keep column titles descriptive enough that the table can be read without guessing the specification

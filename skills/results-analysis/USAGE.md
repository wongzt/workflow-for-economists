# Results Analysis Usage

## Typical Prompts

- "Reconstruct the estimation sample from these do-files and logs."
- "Build a regression-spec matrix from these baseline and appendix tables."
- "Tell me what robustness checks this DID design still needs."
- "Draft the main results text in economics-journal style."
- "Use theory-auditor on these results and write a memo on what to run next."

## Typical Inputs

- `.dta`
- `.do`
- `.log`
- `.tex` regression tables
- `.csv` or `.xlsx` summary files
- variable notes or codebooks

## Typical Outputs

- `analysis-plan.md`
- `regression-spec-matrix.md`
- `table-shells.md`
- `replication-checklist.md`
- optional memo files in `results_memos/`
